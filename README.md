# skadoot.github.io

This repository contains the source code for my personal website, which is hosted at [skadoot.github.io](https://skadoot.github.io).
The website serves as a showcase of my GitHub projects and provides information about myself.

## Project Overview

The main goal of this project is to create and deploy a GitHub Pages website for my GitHub account.
The website is built using HTML and CSS, with a strong emphasis on using semantic HTML and maintaining good HTML structure.
The code was written using the Brackets.io and PhpStorm IDEs.

## Features

- Aesthetic design: The website focuses on delivering a visually appealing and engaging user experience.
- Accessibility: The website is designed with accessibility in mind, aiming to provide an inclusive experience for users with visual and hearing disabilities.
- Semantic HTML: The use of semantic HTML elements ensures proper structure and enhances accessibility.
- Researched font pairs: The website utilizes carefully selected font pairs that are known to work well together, enhancing readability and visual consistency.
- Coordinated colours: The colour palette of the website is thoughtfully chosen to include colour pairs that complement each other, creating a harmonious and visually pleasing design.

## Known Issues

- The currently site has a low-responsive design. In the near future, I plan to add support for mobile devices and other screen types.
- The site currently has low-functionality, meaning testing for accesbility is difficult and at present is not possible to a meaningful extent.
